#### Retail Sales Analytics Dashboard

###### 

###### Overview



This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI.



###### Project Workflow



* Raw data sourced from retail dataset
* Data cleaning and transformation using Python
* Creation of star schema (fact \& dimension tables)
* Data stored in SQL server
* Interactive dashboard built in Power BI



###### Tools Used

* Python (Pandas)
* SQL Server
* Power BI



###### Dashboard



###### Key Insights



* Monthly revenue trends of the year 2023
* Product category performance
* Customer segmentation analysis
